CSharp-Attributeзҡ„зҗҶи§ЈдёҺдҪҝз”Ё

еҸ‘иЎЁдәҺ 2020-05-04 жӣҙж–°дәҺ 2021-02-19 еҲҶзұ»дәҺ C# йҳ…иҜ»ж¬Ўж•°пјҡ

иҝҷж¬ЎиҰҒеӯҰд№ зҡ„жҳҜCSharpдёӯзҡ„еұһжҖ§(Attribute)пјҢз»Ҹеёёеҗ¬и§ҒпјҢд№ҹеңЁUnityдёӯдҪҝз”ЁиҝҮпјҢеҚҙдёҚзҹҘе…¶дёәдҪ•зү©гҖӮеұһжҖ§жҳҜиғҪеӨҹдёәзұ»жҲ–иҖ…жҲҗе‘ҳж·»еҠ йўқеӨ–зҡ„дҝЎжҒҜзҡ„дёңиҘҝгҖӮ

йҰ–е…ҲжҳҜе…Ҙй—Ёж–Үз« зҡ„й“ҫжҺҘпјҡ

  • еұһжҖ§

жҖ»з»“еҠ дёҠиҮӘе·ұзҡ„е®һзҺ°пјҢд№ҹеҸҜд»ҘиҜҙжҳҜжұүеҢ–пјҹиҝҷзҜҮж–Үз« зҡ„жңҖеҗҺжӣҙж–°жҳҜ2011е№ҙпјҢжҜ”иҫғиҖҒдәҶпјҢиҝҳжҳҜиҰҒиҮӘе·ұжЈҖйӘҢдёҖдёӢпјҢиҖҢдё”иҝҳеҠ ж·ұи®°еҝҶгҖӮ

еұһжҖ§(attribute)жҳҜд»Җд№Ҳ

еұһжҖ§жҳҜз”ЁжқҘдёәзұ»жҲ–иҖ…жҲҗе‘ҳж·»еҠ йўқеӨ–дҝЎжҒҜзҡ„дёҖз§ҚдёңиҘҝгҖӮеғҸжҳҜзұ»дёӯзҡ„дҝ®йҘ°з¬Ұpublic,privateйғҪеҸҜд»Ҙи§ҶдёәдёҖз§ҚеұһжҖ§гҖӮ

еҜ№дәҺC++жқҘиҜҙпјҢеғҸиҝҷз§Қдёәзұ»жҲ–жҲҗе‘ҳж·»еҠ йўқеӨ–дҝЎжҒҜзҡ„жғ…еҶөпјҢйңҖиҰҒжҲ‘д»¬еҜ№C++иҝҷй—ЁиҜӯиЁҖиҝӣиЎҢжү©еұ•пјҢйҮҚж–°е®ҡд№үзј–иҜ‘еҷЁгҖӮдҪҶжҳҜCSharpе°ұдёҚйңҖиҰҒиҝҷд№Ҳйә»зғҰгҖӮд№ҹе°ұжҳҜиҜҙпјҢйҖҡиҝҮдҪҝз”Ёеә“дёӯжҸҗдҫӣзҡ„еұһжҖ§пјҢеҸҲжҲ–иҖ…жҳҜиҮӘе·ұж·»еҠ зҡ„еұһжҖ§пјҢеҜ№зј–иҜ‘еҷЁеҸ‘еҮәжҢҮд»ӨпјҢеҸҜд»Ҙз•ҷдёӢдҪҝз”ЁиҖ…жғіиҰҒз•ҷдёӢзҡ„дҝЎжҒҜгҖӮ

C++ гҒӘгҒ©гҒ®ж—ўеӯҳгҒ®иЁҖиӘһгҒ§гҒҜгҖҒгҒ“гҒ®гӮҲгҒҶгҒӘиҝҪеҠ жғ…е ұгӮ’е®ҡзҫ©гҒҷгӮӢе ҙеҗҲгҖҒ иЁҖиӘһд»•ж§ҳиҮӘдҪ“гӮ’жӢЎејөгҒ—гҖҒж–°гҒҹгҒ«гӮігғігғ‘гӮӨгғ©гӮ’дҪңгӮҠзӣҙгҒҷеҝ…иҰҒгҒҢгҒӮгӮҠгҒҫгҒ—гҒҹгҖӮ гҒқгӮҢгҒ«еҜҫгҒ—гҖҒC# гҒ§гҒҜиҮӘеҲҶгҒ§еұһжҖ§гӮ’е®ҡзҫ©гҒ—гҖҒгӮҜгғ©гӮ№гӮ„гғЎгғігғҗгғјгҒ«д»ҳеҠ гҒҷгӮӢгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒҷгҖӮ гҒҷгҒӘгӮҸгҒЎгҖҒгғ©гӮӨгғ–гғ©гғӘгҒ§жҸҗдҫӣгҒ•гӮҢгҒҰгҒ„гӮӢеұһжҖ§гӮ„иҮӘдҪңгҒ—гҒҹеұһжҖ§гӮ’з”ЁгҒ„гӮӢгҒ“гҒЁгҒ§гҖҒ гӮігғігғ‘гӮӨгғ©гҒ«еҜҫгҒҷгӮӢжҢҮзӨәгӮ’иЎҢгҒЈгҒҹгӮҠгҖҒгӮҜгғ©гӮ№гҒ®еҲ©з”ЁиҖ…гҒ«еҜҫгҒҷгӮӢжғ…е ұгӮ’ж®ӢгҒҷгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒҷгҖӮ

дҪҝз”ЁеұһжҖ§зҡ„еңәеҗҲпјҡ

  • жқЎд»¶еҲҶж”Ҝзј–иҜ‘пјҢеҗ‘зј–иҜ‘еҷЁеҸ‘еҮәжҢҮд»Өзҡ„ж—¶еҖҷдҪҝз”ЁпјҲConditionalе’ҢObsoleteзӯүпјү
  • дҪңиҖ…зҡ„дҝЎжҒҜд»Ҙmeta dataзҡ„еҪўејҸжҸ’е…ҘеҲ°зЁӢеәҸеҪ“дёӯпјҲAssemblyTitleзӯүпјү
  • еҲ©з”ЁеҸҚе°„пјҢзЁӢеәҸиҝҗиЎҢж—¶еҸ–еҮәжғіиҰҒзҡ„еұһжҖ§жғ…жҠҘ

Pointпјҡ

  • CSharpзҡ„иҜқпјҢеҸҜд»ҘиҮӘз”ұзҡ„дёәзұ»жҲ–иҖ…жҲҗе‘ҳж·»еҠ иҮӘе®ҡд№үзҡ„еұһжҖ§гҖӮ
  • дёҖйғЁеҲҶзҡ„еұһжҖ§пјҢеҸҜд»ҘдҪңдёәеҜ№зј–иҜ‘еҷЁжҲ–иҖ…VisualStudioеҸ‘еҮәжҢҮд»Өзҡ„ж–№ејҸдҪҝз”ЁгҖӮ
    • [Obsolete] class OldClass {} вҖҰ еҸӨгҒ„гғҗгғјгӮёгғ§гғігҒЁгҒ®дә’жҸӣжҖ§гҒ®гҒҹгӮҒгҒ гҒ‘гҒ«ж®ӢгҒ—гҒҰгӮӢгҒ‘гҒ©гҖҒгҒ“гҒ®гӮҜгғ©гӮ№гҒҜгӮӮгҒҶдҪҝгӮҸгҒӘгҒ„гҒ§гҖӮ
    • [EditorBrowsable] T Property; вҖҰ Visual Studio гҒ® IntelliSenseпјҲгҒӘгҒ©гҒ®гҖҒй–Ӣзҷәгғ„гғјгғ«гҒ®иЈңе®Ңж©ҹиғҪпјүгҒ§иЎЁзӨәгҒҷгӮӢгҒӢгҒ©гҒҶгҒӢгӮ’иЁӯе®ҡгҒ—гҒҫгҒҷгҖӮ

дёҠйқўзҡ„дёӨдёӘдҫӢеӯҗжҲ‘дёҚеӨӘзҗҶи§ЈпјҢйҒҮи§ҒеҶҚиҜҙеҗ§гҖӮ
